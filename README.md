# キャビネット収容計算アプリ

PC機器の収納に必要なキャビネット台数を自動計算するWebアプリケーションです。

## 🌟 機能

- ✅ 50行のPC機器入力フォーム
- ✅ リアルタイム容積計算
- ✅ 必要キャビネット台数の自動表示
- ✅ データ自動保存機能（LocalStorage）
- ✅ モダンで美しいダークモードUI
- ✅ レスポンシブデザイン対応

## 📦 キャビネット仕様

- **内寸サイズ**: 455mm × 690mm × 1260mm
- **容積**: 395,577,000 mm³

## 🚀 使用方法

1. `index.html` をブラウザで開く
2. PC機器の情報を入力：
   - 機器名称
   - 縦（mm）
   - 横（mm）
   - 高さ（mm）
   - 台数
3. 自動的に必要なキャビネット台数が表示されます
